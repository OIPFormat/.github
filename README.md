# OIPFormat

Open specifications for **OIPF — the Operational Intelligence Packaging Format**:
Operations-as-Code for packaging operational-intelligence use cases as versioned, testable,
auditable artifacts that deploy across sites by configuration.

- **[OIPF-Specs](https://github.com/OIPFormat/OIPF-Specs)** — specification, JSON schemas, use-case catalogue, and the reference example.

Sibling open standards: UAPF (algorithm packaging), MCPF (agent trust).
Apache-2.0. Draft v0.3.0.
